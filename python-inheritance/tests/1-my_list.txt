The ``MyList`` class inherits from ``list`` and adds a sorting display method.

Initialize the class and verify inheritance:
    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

Test appending an element:
    >>> my_list.append(1)
    >>> print(my_list)
    [1]

Verify sorting of basic positive integers:
    >>> my_list = MyList([1, 4, 2, 3, 5])
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Verify sorting with negative integers:
    >>> my_list = MyList([10, -5, 0, 2])
    >>> my_list.print_sorted()
    [-5, 0, 2, 10]

Verify behavior with an empty list:
    >>> my_list = MyList([])
    >>> my_list.print_sorted()
    []

Verify sorting with duplicate elements:
    >>> my_list = MyList([2, 1, 2, 1])
    >>> my_list.print_sorted()
    [1, 1, 2, 2]
